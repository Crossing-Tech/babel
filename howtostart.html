How to start a Babel project¶

Using Maven¶

To add Babel Camel in a Maven project, just add the following dependencies to your pom.xml file:

<dependency>
  <groupId>io.xtech.babel</groupId>
  <artifactId>babel-camel-core</artifactId>
  <version> 0.6.0-SNAPSHOT </version>
</dependency>

If you also want to use the Babel Camel Mock extension for your tests, you may also add:

<dependency>
  <groupId>io.xtech.babel</groupId>
  <artifactId>babel-camel-mock</artifactId>
  <version> 0.6.0-SNAPSHOT </version>
  <scope>test</scope>
</dependency>

Using Sbt¶

To add Babel Camel in a Sbt project, you may just add the following dependencies to your build configuration:

libraryDependencies += "io.xtech.babel" % "babel-camel-core" % "0.6.0-SNAPSHOT"
//if you want to use the mock keyword to simplify your tests:
libraryDependencies += "io.xtech.babel" % "babel-camel-mock" % "0.6.0-SNAPSHOT" % "test"

Babel Archetypes¶

The Babel Camel Archetype may help you to start a maven project using Babel. Currently, you need to first install the Babel project with Maven following the Build Babel with Maven part. Once you have installed the babel project locally, use the following command to start a new maven project including Babel through the following command:

mvn archetype:generate                       \
-DarchetypeGroupId=io.xtech.babel            \
-DarchetypeArtifactId=babel-camel-archetype
