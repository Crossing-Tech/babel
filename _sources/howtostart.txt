
How to start a Babel project
============================

Babel Archetypes
++++++++++++++++


The Babel Camel Archetype may help you to start a maven project using Babel. Once you have installed the babel project locally, use the following command to start a new maven project including Babel::

  mvn archetype:generate                          \
  -DarchetypeGroupId=io.xtech.babel            \
  -DarchetypeArtifactId=babel-camel-archetype
